# XQGrandPopupView
GrandPopupView的OC版。GrandPopupView是一个自定义程度非常高的弹窗组件。它对外提供一个容器视图，因此开发者可以完全自定义弹窗内容。同时GrandPopupView封装了弹窗的显示与隐藏，并提供了两种默认的动画效果，子类可通过重写的方式扩展动画效果。

# 集成

```
pod 'XQGrandPopupView'
```

# 注意

弹窗推荐继承GrandPopupView。
